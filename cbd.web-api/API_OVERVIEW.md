# CBD Diary API Overview

## Созданные модули

### 1. CBT Module (КПТ модуль)

**Эндпоинты:** `/api/v1/cbt`

Основная функциональность дневника КПТ (Когнитивно-поведенческая терапия):

- `POST /cbt/entries` - Создание новой записи дневника
- `GET /cbt/entries` - Получение списка записей с фильтрацией и пагинацией
- `GET /cbt/entries/:id` - Получение конкретной записи
- `PATCH /cbt/entries/:id` - Обновление записи
- `DELETE /cbt/entries/:id` - Удаление записи
- `GET /cbt/entries/tags/all` - Получение всех тегов пользователя
- `PATCH /cbt/entries/:id/mood-after` - Обновление настроения после записи

**Структура записи:**

- Ситуация (что произошло)
- Цепочки мыслей с эмоциями и интенсивностью
- Реакции и поведение
- Оценка настроения до/после
- Теги и метаданные

### 2. Sync Module (Модуль синхронизации)

**Эндпоинты:** `/api/v1/sync`

Синхронизация данных между мобильным приложением и сервером:

- `POST /sync/user-data` - Основная синхронизация данных
- `POST /sync/resolve-conflicts` - Разрешение конфликтов синхронизации
- `GET /sync/status` - Статус синхронизации пользователя
- `PUT /sync/force-sync` - Принудительная пересинхронизация
- `GET /sync/health` - Проверка работоспособности сервиса

**Возможности:**

- Двунаправленная синхронизация
- Автоматическое обнаружение и разрешение конфликтов
- Инкрементальные обновления
- Офлайн-поддержка

### 3. Analytics Module (Модуль аналитики)

**Эндпоинты:** `/api/v1/analytics`

Аналитика и статистика пользователей:

- `GET /analytics/user-stats` - Основная статистика пользователя
- `GET /analytics/emotions` - Анализ эмоций за период
- `GET /analytics/mood-trends` - Тренды настроения
- `GET /analytics/cognitive-insights` - Когнитивные инсайты и паттерны
- `GET /analytics/progress-report` - Отчет о прогрессе с достижениями
- `GET /analytics/summary` - Полная сводка аналитики
- `PUT /analytics/refresh-stats` - Обновление статистики

**Аналитические возможности:**

- Статистика активности (серии дней, общие записи)
- Анализ эмоций и их паттернов
- Тренды настроения во времени
- Когнитивные искажения и их частота
- Эффективность КПТ техник
- Персональные рекомендации
- Система достижений

## Технологии и архитектура

### База данных (PostgreSQL + Prisma)

- **Users** - пользователи
- **CbtEntry** - записи дневника КПТ
- **Emotion**, **EmotionCategory** - эмоции и их категории
- **SyncOperation** - операции синхронизации
- **UserStats** - предвычисленная статистика
- **Translation** - многоязычность

### Безопасность

- JWT аутентификация
- Изоляция данных пользователей
- Валидация входных данных
- CORS настройки

### Особенности

- Swagger документация (`/api/docs`)
- Типизированные DTO с валидацией
- Пагинация и фильтрация
- Обработка ошибок
- Поддержка часовых поясов

## Совместимость с мобильным приложением

API полностью совместим с существующим Vue/Tauri мобильным приложением:

1. **Структура данных** соответствует схеме из `AddEntry.vue`
2. **Синхронизация** поддерживает офлайн-режим мобильного приложения
3. **Аналитика** предоставляет данные для дашборда в приложении
